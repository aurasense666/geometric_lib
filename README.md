## Тестирование

В библиотеке есть папка `tests`, в которой расположен набор юнит‑тестов для проверки корректной работы функций вычисления площадей и периметров фигур.

Команды для запуска тестов:
-всех тестов: python run_tests.py или python -m unittest discover tests -v

-по отдельности: python -m unittest tests.test_circle -v, python -m unittest tests.test_square -v,
python -m unittest tests.test_rectangle -v, python -m unittest tests.test_triangle -v

### Описание тестов

- Проверяют корректность вычисления площади круга, квадрата, прямоугольника и треугольника.  
- Проверяют корректность вычисления периметра (длины окружности) для всех фигур.  
- Проверяют различные варианты входных данных: целые числа, дробные значения, нули, а также достаточно большие положительные значения.
